# 概率论与数理统计

## 概要
> 披着羊皮的狼

这门课分为两个大的方面：概率论、数理统计。概率论主要涉及随机事件和概率、随机变量及其概率分布、数字特征、大数定律。数理统计主要涉及统计与抽样分布、估计、假设检验。是一门综合性和实践性很强的课程。

笔者在刚接触这门课的时候，一听“概率论与数理统计”这个名字，以为全是高中知识，十分简单。学到随机变量及其概率分布时就感觉不太对劲，学到后面估计和假设检验真是烟斗不待言了，进度和坐过山车一样快，要记得东西和山一样多。到了后来，笔者上课就只是机械性的把知识点全部记在电脑上，脑子里是一点没留。总而言之，这门课肯定比你想象中的难度要高。
