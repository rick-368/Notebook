# 普通物理学Ⅱ

## 概要
> 撒旦也不会学普通物理学Ⅱ

普物Ⅱ讲解电磁学、光学和量子力学。具体而言，电磁学是以麦克斯韦方程组为基础，光学讲解几何光学（反射折射以及透镜等）、波动光学（干涉衍射偏振等）以及基本的光的量子性，量子力学讲解薛定谔方程、不确定性原理，以及量子阱、量子隧穿、氢原子模型等等。

这门课我不想过多评价，只能说其没有从课改中删去绝对是一大败笔。它不仅考察学生的物理素养，对数学能力的要求也是极高的（其中涉及微分方程的求解等等），如果不是有聪慧过人的大脑或是物理竞赛的底子，那上课基本和听天书无异（尤其是到量子力学部分）。过于逆天的难度让人望而却步，每一个经历过这门课的学生都对它“赞誉有加”，在本届的期中考试后甚至引发了“弃修潮”（防止被挂）。当觉得生活很如意的时候，不妨来学学这门课。

> 在TuringCourses上，普通物理学Ⅱ超越了数学分析和线性代数，成为唯一一个难度拿满五颗星的数理基础课程。
