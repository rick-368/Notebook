# 无线网络应用

## 概要
> 贪心不足蛇吞象

这门课主要介绍了无线网络的基础知识，如网络通信、网络寻址、无线技术等等。

课程内容及其硬核，但由于它是通识课，所以这反而成为了它的缺点。每次课老师都会滔滔不绝的讲四节课，但内容却既专业又枯燥，很难让人听下去。如果认真学习应该能学到很多知识，但笔者也仅仅坚持了四节课就放弃听讲了。不过课程的实验不算难，有详细的步骤指导，而且笔者摊上了一个好的小组，基本不用笔者干什么活实验就弄完了，最后给分还是挺不错的。
