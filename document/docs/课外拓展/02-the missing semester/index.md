# The Missing Semester

## 概要

“这门课将会教会你许多大学的课堂上不会涉及但却对每个 CSer 无比重要的工具或者知识点。例如 Shell 编程、命令行配置、Git、Vim、tmux、ssh 等等。” ————CS自学指南

这门课是笔者真正自学的开始。笔者认为，学这门课是及其富有成就感的，仅仅花费40分钟时间，你就能了解并入门Git、Vim等工具，并掌握一些命令行技巧，而这些知识对于计算机学习乃至将来工作都是及其重要的。（唯一的一点是笔者现在还是不习惯使用Vim的反人类键位，更习惯于用鼠标戳戳点点:smile:）
