# 面向对象程序设计

>大翁老师的残躯

本节课主要以C++为例，学习面向对象语言的特点。主要涉及C++的基础语法、容器、类、函数复用（继承、多态、运算符重载）、异常等。

根据图灵网站，翁恺老师在 21 级风评飞速下跌，大有晚节不保的情况。但大翁老师的授课水平是毋庸置疑的（和某门数据库课程的老师形成了鲜明对照），翁恺老师除了讲解大纲要求的C++98 标准的内容，还会向我们拓展一些C++11的内容。并且他在授课过程中，会结合一些实际案例，帮助我们更好地理解面向对象编程的思想和方法。
