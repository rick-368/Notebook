# 数据库系统

## 概要
> 上千页不明所以的课件，以及只会棒读的老师

本课程分为两大部分：数据库应用和数据库系统的实现。数据库应用主要涉及数据库的一些基本概念、关系模型、SQL语言、ER模型。而数据库系统的实现主要涉及关系型数据库设计，存储和文件结构，索引，查询处理与优化，事务与并发控制，故障恢复。

图灵精品专业基础课中不得不吃的一环。首先是不合理的课程设置，据说以前这门课是两门课，后来合并了，导致课程量爆炸，十几章总共合起来有近千页ppt，根本复习不完。其次是敷衍了事的老师，一整条两个小时的课，全部都在用一种平铺直叙、让人混混欲睡的粘腻语调棒读ppt，这是唯一一门我就算听智云也听不下去的课程。我认为他在讲台上的作用不如一直草履虫，他甚至不如让我们自己看三节课的ppt，至少我不会听到他那闷闷的、让人烦躁的腔调。然后就是如儿戏一般的作业。一共六个实验，前面四个实验全都使用着比我还老的ppt，且我都没看出有任何设置的意义，只是纯粹浪费时间。后面的第一个大实验是设计一个图书管理系统，隔壁的孙建伶老师班有详细的JDBC框架，而我们甚至连个框架都没有，纯粹手搓，绝对匠人精神。

当然这门课也不是没有一点可取之处。最后一个大实验十分有意思。基于 CMU 15-445 的 BusTub 框架，让我们分组实现一个基于C++的MiniSQL。由于是CMU留下的瑰宝，在加上隔壁孙老师的努力，框架和实验指导还是比较完善且清晰的。且这个项目也有一定难度，对于初步认识大工程、提升代码编写能力、应用 C++ 有很大帮助。
