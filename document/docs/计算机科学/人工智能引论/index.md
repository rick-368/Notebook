# 人工智能引论

## 概要
> 囫囵吞枣

本课程对人工智能的基础知识进行了概略性的介绍，包括表达与推理、搜索、深度学习、强化学习、博弈论等内容。课程内容很多，基本上是各式各样的概念、算法和公式。

笔者上的是朱霖潮老师的课，上课节奏较慢，虽然笔者已经感受到老师想尽力带动课堂气氛了，但授课的内容难免枯燥乏味。许多内容都是以概念性的形式呈现，听完一节课脑子空空。虽说讲了很多新奇的理念，但很多都是浅尝辄止，课后的实验作业只是形式主义，最后的考察也基本上以背诵为主。
