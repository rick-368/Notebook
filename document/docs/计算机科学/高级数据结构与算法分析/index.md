# 高级数据结构与算法分析

## 概要
> 计算机专业课不让用电脑

此课主要介绍一些高级数据结构（如AVL树、红黑树、二叉堆、斜堆）等，以及一些算法（如回溯、分治、动态规划、贪心算法、P与NP、近似算法、局部搜索、随机算法、并行算法、外部排序）的理论和应用。

此课理论性较强，推导证明的东西很多，如摊还分析等很是头疼，也是大二上计算机专业课的难点之一。毛宇尘老师上课讲解很细致，且全程板书，十分负责，给分也十分慷慨，就是上课不让用电脑一事让笔者十分不能理解，这对于用电脑记笔记的同学来讲非常不友好（有很多节课我都是将笔记以markdown的格式手敲在手机备忘录里的）
