# 计算机系统Ⅲ

## 概要

> “指环王：王者归来”

本课作为系统贯通系列的第三门课程，也是三部曲的终章了。前两章是计算机体系结构，包括进行量化研究的一些公式，如阿姆达尔定律等等，还有静态和动态调度流水线。然后计算机组成部分主要是对cache的讲解。而操作系统部分则讲了内存（虚拟内存部分）、文件系统、I/O 等。最后涉及到部分系统安全以及SIMD和MIMD两种并行计算架构方面的知识。

自然，实验仍是本课程的重点。前两个实验是给系统二实现的CPU添加动态分支预测和cache模块，接下来是三个软件实验，包括虚拟内存、用户态以及 page fault 和 fork 的实现等。最后是组队完成的 X part实验。

X part实验是最有难度的部分，必做部分要求在硬件层面实现MMU（使用系统二的CPU即可），并在此CPU上运行起 lab5 中编写的 kernel。这一部分占60分。而剩余40分则是有多个可选方向，以来完善自己的软件和硬件功能，如在 MMU 基础上添加 Cache 模块（40'），实现内核的输入输出(20')，实现shell（10'~40'）等等。最终两个部分加起来到100即可（如果只做拓展部分则最多是80分）。

笔者在做X part实验时，由于摊上了一个好teammate（外加chatGPT的帮助），笔者只帮助调试了一天便神奇的实现了MMU（当时激动的和Niko夺冠差不多的场面）。后面的拓展部分也是挑了比较简单的软件拓展（shell、内核的输入输出和malloc和free函数的实现），所以并没有花太大功夫。也算是给这门纵贯三个学期的课程画上了一个圆满的句号吧！（当然笔者写这一部分的时候还没考试，希望老师手下留情~）。
