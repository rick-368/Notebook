# 计算机系统II

## 概要
> 软硬兼施不可取

本课程沿袭计算机系统一，主要分为计算机组成和操作系统两部分，即硬件和软件两个部分。硬件部分的重点是RISC-V流水线CPU的设计，软件部分包括进程、线程、调度、同步等内容。

此课最主要的部分还是在实验。要求在系统一的单周期cpu的基础上实现五级流水线cpu，之后还要实现CSR流水线以及进行陷入检测和处理，最后在cpu上运行自己编写的Mini Kernel。硬件实验工作量大，调试困难，十分考验人性（由于一个小信号设置异常调试一整天是十分正常的）；虽说软件实验大部分较为简单，但也需要花费大量时间在思考题上。

这部分笔记笔者只记了软件部分的内容（硬件课大部分时间在摸鱼）。
